## Comandos para configurar o github inicialmente 

- git init
- git config --global user.name "<SEU_NOME_USUARIO>" 
- git config --global user.email "<SEU_EMAIL>"
- git clone https://github.com/drlink/projeto-horus.git
- git remote add origin https://github.com/drlink/projeto-horus.git
- git branch -M main
