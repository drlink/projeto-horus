# projeto-horus
Repositorio para centralizar a documentação do projeto de cyber lab da faculdade impacta  

## 1 - Orientações para configurar o repositório.

## comandos para configurar o github inicialmente 

- git init
- git config --global user.name "<SEU_NOME_USUARIO>" 
- git config --global user.email "<SEU_EMAIL>"
- git remote add origin https://github.com/drlink/projeto-horus.git
- git branch -M main
- git add . => adiciona o conteudo modificado 
- git commit -m "<SUA_MENSSAGEM>"
- git push -u origin main
